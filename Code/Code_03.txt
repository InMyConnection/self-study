Глава про шрифт Брайля. Азбука для слепых представлена шестью двоичными символами. В качестве двоичного символа выступает точка, которая либо есть, либо отсутствует.
Из шести двоичных символов можно составить 64 комбинации.
Для увеличения возможностей передачи информации с помощью этих 64 комбинаций, существуют коды переключения (shift) и escape-коды.
Код переключения меняет смысл всех следующих кодов, пока его действие не будет отменено.
Escape-код указывает, что только один следующий за ним код будет интерпретироваться по-другому.
