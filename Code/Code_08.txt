В этой главе рассматривается сначала восьмеричная система счисления, способ перевода чисел из десятичной в восьмеричную, сложение и уможение восьмеричных чисел.
Потом также описывается четверичная система и двоичная.
Система счисления, основанная на двух цифрах является предельно упрощенной, что позволяет использовать её для описания электрических явлений. Включено что-то - это 1. Выключено - 0.
Bit - binary digit (двоичная цифра).
