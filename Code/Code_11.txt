Для проектирования цепей с переключателями можно применять средства булевой алгебры.
Логический вентиль - несколько реле, включенных в схему параллельно или последовательно, которые решают простейшую логическую задачу.
Преимущество реле по сравнению с переключателями в том, что их можно включать и выключать, используя другие реле, а не вручную. Выход одного реле может подключаться к выходу другого.
У двухпозиционного реле (double-throw) два электрически противоположных выход: когда на одном есть напряжение, на другом его нет.
Из последовательно соединенных реле получается вентиль И. У вентиля И может быть больше двух выходов. При другом подключении выходов реле получаем вентиль ИЛИ-НЕ.
Из двух реле, соединенных параллельно, получается логический вентиль ИЛИ. У вентиля ИЛИ может быть больше двух выходов. При другом подключении выходов реле получаем вентиль И-НЕ.
Из одиночного реле можно сделать инвертор.
Выход одного вентиля (или инвертора) может быть входом одного или нескольких вентилей (или инверторов).
Выходы двух или более вентилей (или инверторов) никогда не соединяются между собой.
Схема, состоящая из двух инверторов и четырех вентилей И называется дешифратором 2 линии на 4 (2-Line-To-4-Line Decoder).
Устройство из обычного реле называется буфером (buffer).
