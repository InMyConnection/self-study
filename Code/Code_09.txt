Один бит информации - это минимально возможное её количество. Более сложную информацию можно передать в нескольких битах.
Информацию можно рассматривать, как выбор между двумя и более возможностями.
В теории коммуникаций есть понятие избыточной надёжности (redundancy), необходимой для противодействия шуму (noise).
Чем больше битов мы используем, тем больше имеем досупных вариантов для кодирования информации. Каждый дополнительный бит удваивает число доступных кодов.
В своей основе биты - это числа. Для представления информации в виде битов достаточно пересчитать количество доступных возможностей. Оно определяет сколько битов понадобится для того, чтобы присвоить каждой возможности уникальный номер.
