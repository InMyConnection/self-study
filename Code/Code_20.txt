Набор кодированных символов (coded character set) - система кодов для буквенно-цифровых (alphanumeric) символов.
Американский стандартный код для обмена информацией (American Standart Code for Information Interchange, ASCII) - 7-битовая кодировка.
ASCII включает цифры, знаки препинания, прописные и строчные буквы латинского алфавита и группу неотображаемых символов, использующихся для выполнения каких-либо действий в телетайпных аппаратах.
Обёъм памяти в байтах, занятый текстом в кодировке ASCII, равен количеству символов в тексте.
Кодировка Unicode - 16-битовая. Количества кодов в ней достаточно для кодирования любых языков мира.
Первые 128 символов Unicode совпадают с символами ASCII.
В кодировке Unicode один символ текста занимает 2 байта.
