Защелка, применяемая для текущего хранения суммы называется аккумулятором (accumulator).
Коды команд (instruction code), или коды операций - численные коды, используемые специальным образом.
Управляющие сигналы (control signals) - сигналы, которые руководят работой всех компонентов.
Процесс извлечения команды из памяти называется выборкой команды (instuction fetch).

Итак, можно собрать примитивный компьютер из счетчика (program counter), селектора, массива памяти и трех защелок.
Команды типа "Перейти к определенной ячейке памяти", а не выполнять последовательно, называют командами ветвления (branch).
Условный переход (conditional jump) - переход, происходящий лишь при выполнении определенного условия.
Цифровые (digital) устройства работают с дискретными цифрами. Дискретные данные принимают набор фиксированных значений. Аналоговые (analog) данные непрерывны и могут принимать любое значение из заданного интервала.
Цифровой компьютер состоит из 4 основных компонентов: процессор, память, устройства ввода и вывода.
Центральное процессорное устройство (Central Processor Unit, CPU) имеет разрядность, равную ширине большиснства своих каналов данных.
Аппаратное обеспечение (hardware)  программное обеспечение (software).
В компьютерных программах различают код (code), т. е. команды, и данные (data).
Коды команд, на основании которых процессор выполняет какое-то действие называют машинными кодами или машинным языком.
Для обозначения команд используют мнемонические коды. Аргументы конкретизируют выполнение команды.
При написании кодов для обозначения ячеек памяти можно пользоваться метками (labels).
Язык программирования, использующий мнемонические коды, называют языком ассемблера (assembly language). Каждому оператору языка ассемблера соответствует свой числовой код.
