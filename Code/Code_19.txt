Первые серьёзные модели процессоров: Intel 8080 и Motorola 6800. Эти микросхемы являются однокристальными процессорами (single-chip).
Intel 8080 имеет 40 выводов: 3 электропитания, 1 земля, 2 входных синхронизирующих сигнала, 16 выходных для адресации памяти, 8 контактов на вход и выход для обмена данными, 10 контактов под управляющие сигналы.
Кроме аккумулятора в 8080 есть 6 регистров в качестве сверх-быстрой памяти.
Считывани байта команды называется выборкой команды (instruction fetch).
Команды используются для передачи данных из памяти в процессор. Адрес ячейки может быть указан прямой адресацией (direct addressing), индексной (indexed), непосредственной (immediate).
Все флажки состояния (перенос, знак, ноль четность...) хранятся в 8-битовом регистре - слове состояния программы (program status word).
Память бывет с произвольным доступом, последовательным доступом (sequential access), стеком (stack). Помещение информации в стек называется вталкиванием (push), извлечение - выталкиванием (pop)ю
Для стека выделяется область памяти, не занятая другими программами. Для адресации этой области используется указатель стека (stack pointer).
Из-за ошибки в программе может возникнуть переполнение стека (stack overflow) или исчезновение стека (stack underflow).
Для обмена информацией процессора с переферийными (peripheral) устройствами адреса внешним устройствам могут выделяться из обычного адресного пространства - это ввод-вывод с распределением памяти (memory mapped input/output).
Если используются дополнительные адреса, то они назывются портами ввода-вывода (input/output ports).
Механизм прерываний (interrupts) позволяет процессору получать сигналы от внешних устройств сразу.
Программа-эмулятор по одному просматривает все машинные коды программы, написанной для одного процессора, и выполняет соответствующее действие на другом.
Конвейеризация (pipelining) - выполняя одну инструкцию, процессор заранее считывает следующую.
Кэш (cache) - сверх-быстрая память внутри процессора, хранящая недавно выполнявшиеся команды.
