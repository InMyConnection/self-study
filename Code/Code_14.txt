Тут про зуммер (buzzer), который похож на инвертор. Если убрать переключатель получится вибратор (oscillator).
Цикл колебаний - это интервал, в течение которого выход вибратора изменяется, а затем возвращается в исходное состояние.
Время, которое занимает один цикл, называется периодом колебаний.
Величина колебаний, обратная периоду, называется частотой колебаний. Измеряется в Гц (число колебаний в секунду).
Обратная связь (feedback). Схема триггера (flip-flop). Триггер сохраняет информацию. Существует несколько разновидностей триггеров. Простейший - RS-триггер (Reset-Set).
RS-триггер состоит из двух вентилей ИЛИ-НЕ с обратной связью. Имеет входы Reset, Set и выходы Q b не Q.
D-триггер со срабатыванием по уровню (level-triggered D-type flip-flop) состоит из инвертора, двух вентилей И и RS-триггера. Имеет два входа: данные (data) и синхронизация (clock).
Иногда его ещё называют защелкой (latch).
D-триггер со срабатыванием по фронту собирается из двух блоков. Выходы меняются только при изменении сигнала Clock с 0 на 1, то есть совершает положительный переход.
Делитель частоты (frequency divider) собирается из вибратора и D-триггера, срабатывающего по фронту, с обратной связью.
Из нескольких делителей частоты можно собрать сквозной щетчик (ripple counter).
