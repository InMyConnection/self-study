Придумываем схему работы двух фонариков на расстоянии. Потом реализуем соединение с общим проводом (common) для экономии проводов.
Если мы имеев дело с большим напряжением и дальним расстоянием, то в качестве общего провода может выступать земля. Для этого нужно реализовать электрический контакт с землей - заземление.
Длина, на которую мы можем протянуть провода, ограничена сопротивлением проводника.
