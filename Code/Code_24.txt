Язык ассемблера считается языком низкого уровня (low-level), поскольку он напрямую взаимодействует с оборудованием компьютера. Остальные языки программирования - высокого уровня (high-level).
Для языка нужен синтаксис - правила построения выражений - и компилятор (compiler) - программа, преобразующая операторы языка в машинные коды.
Оператор языка высокого уровня превращается во множество машинных команд.
Языки высокого уровня в основном переносимы (portable), то есть одинаково хорошо работают с разными процессорами.
Переменная (variable), присваивание (assignment), цикл (loop), блок кода (block), условные структуры.
Компилятор считывает файл с исходной программой целиком, а затем создает исполняемый файл.
Интерпретатор считывает программу оператор за оператором и сразу выполняет их. При этом исполняемый файл не создается.
