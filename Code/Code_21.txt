Шина (bus) - набор цифровых сигналов, подаваемых на все платы компьютера. Сигналы бывают: адресные, вывода данных, ввода данных, управляющие. Шина отвечает за подчу питания на компоненты компьютера.
На материнской (системной) плате (motherboard) располагаются гнёзда для вставки разъёмов плат - слоты расширения (expansion slots), вставляемые платы - платы расширения (expansion cards).
Открытая архитектура (open architecture) - архитектура, на которую опубликованы спецификации, что позволяет другим производителям создавать дополнительное оборудование на устройства с такой архитектурой.
Выходной сигнал с тремя состояниями (3-state) - логический 0, логическая 1 и полное отсутствие сигнала.
Статическая память (static, SRAM) и динамическая (dynamic, DRAM). Динамической требуется периодическая регенерация (refresh).
Видеокарта (видеоадаптер) - аппаратный компонент, управляющий работой монитора.
Отдельные элементы изображения на мониторе - пикселы (pixel - "picture element").
Предельная скорость реакции экрана на изменившееся напряжение - полоса пропускания (bandwidth). Разрешение (resolution) видеоадаптера.
Графический (graphics) видеоадаптер способен отображать не только текст.
Жесткий диск состоит из нескольких металлических дисков. Поверхность диска разделена на концентрические кольца - дорожки (tracks), каждая разделена на несколько дуг - секторов (sectors).
Оперативная память энергозависима в отличие от накопителя. Адресные выходы процессора используются только для обращения к памяти.
Чтобы с данными мог работать процессор, их нужно перенести из накопителя в оперативную память.
